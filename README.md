# Ecommerce-Sales-Analytics
Overview

This project analyses online retail sales to understand customer behaviour, revenue trends, and product performance.
The goal is to uncover opportunities for improving sales, customer retention, and operational efficiency.

This project demonstrates skills in:

Data cleaning and preparation

Customer and sales analytics

RFM segmentation

SQL querying for business questions

Power BI dashboard development
